KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a modern friendship management web app built with Next.js. It helps users track friends, manage interactions, and maintain meaningful relationships through reminders and timelines.

🚀 Live Demo

https://your-live-link-here.com

📌 Features
👥 Display friends from JSON data
📄 Friend details page with full profile
📅 Track days since last contact
🔔 Quick check-in actions (Call, Text, Video)
⚡ Timeline to record interactions
🔍 Search-based timeline filtering
📊 Stats page with analytics (Pie chart)
📱 Fully responsive design (mobile, tablet, desktop)
❌ Custom 404 page
⏳ Loading spinner while fetching data
🔔 Toast notifications for interactions
🛠️ Technologies Used
Next.js (App Router)
React.js
Tailwind CSS
DaisyUI
React Icons
React Toastify
Recharts
